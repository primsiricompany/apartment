วาง URL ของ Google Apps Script ที่ได้จากการเผยแพร่ Web App

ขั้นตอนการเชื่อมโยงระบบเข้ากับ Google Sheets

โปรดปฏิบัติตามขั้นตอนด้านล่างนี้เพื่อเชื่อมโยงฐานข้อมูลการเช่ากับคลาวด์ของคุณ:

  1. ไปที่ Google Sheets และสร้างสเปรดชีตใหม่
  2. ตั้งชื่อแผ่นงาน (Sheet) แผ่นแรกว่า Data
  3. ไปที่เมนู ส่วนขยาย (Extensions) > Apps Script
  4. คัดลอกรหัสโค้ด Apps Script (ในกล่องข้อมูลด้านล่างนี้) ไปใส่ในแถบตัวแก้ไขโค้ด
  5. คลิกปุ่ม การทำให้ใช้งานได้ (Deploy) ด้านบนขวา > เลือก การทำให้ใช้งานได้ใหม่ (New deployment)
  6. เลือกประเภทเป็น เว็บแอป (Web app) ปรับค่าผู้มีสิทธิ์เข้าถึงเป็น ทุกคน (Anyone) และกด Deploy
  7. คัดลอก URL ของเว็บแอปที่ได้ นำมาวางลงในช่อง "Web App URL" ในแผงตั้งค่าด้านซ้าย แล้วกดเชื่อมต่อและทดสอบ
